# gram

_gram_ is a package for simulating TinyRAM (http://www.scipr-lab.org/specs.html) written in GO.


## build

```bash
go get github.com/LayerXcom/gram
cd $GOPATH/src/github.com/LayerXcom/gram
go build .
```

## usage
```bash
./gram <path/to/tinyram.asm>
```
